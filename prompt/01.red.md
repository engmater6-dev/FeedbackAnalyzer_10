## RD-03 | 테스트 플랜 샘플 예제 1개 선정

### 프롬프트 (원문)

```
분석한 내용을 바탕으로
테스트 플랜 작성을 위한 샘플 예제 1개를 선택해줘.

선택한 예제에 대해 다음을 명시하라:
- 선택한 기능 이름
- README 요구사항 번호 (예: 기본 요구사항 4번)
- 입력 예시
- 기대 출력
- 이 예제를 선택한 이유 2줄

코드 작성 금지. 선택 결과 문서만 출력.
```

### 답변 (축약)

- **기능**: PRD 대표 시나리오 — 감정·카테고리 분석 및 복합 필터.
- **README**: 주요 기능 **4번**(필터링).
- **입력**: `"배송이 너무 늦어요. 화가 납니다."` / 필터 `부정`+`배송`.
- **기대**: 부정 1, 배송 1, filter ≥1건.
- PRD·Mom Test·CODE_SMELL가 동일 케이스로 겹침.
