## 4. Red Gate 실행 결과

**명령** (`src/python`):

```bash
pytest tests/domain/test_anchor_prd_example.py tests/domain/test_filters_regression.py -v
```

**결과**: `4 failed, 2 passed` → **Red 완료** (의도적 실패 확인)

| 테스트 | 결과 | 검증 버그 |
|--------|------|-----------|
| `test_sent_classifies_anchor_as_negative` | **FAIL** | B-01 / `화가`≠`화남` (S-T04) |
| `test_filter_negative_and_shipping_returns_at_least_one` | **FAIL** | B-01, B-02 (S-F02, S-F03) |
| `test_sent_neutral_count_equals_filter_neutral_count` | **FAIL** | B-01 (S-F01) — 중립 2 vs 필터 1 |
| `test_kw_category_count_equals_filter_category_count` | **FAIL** | B-02 — `품질` kw 1 vs filter 0 |
| `test_kw_classifies_anchor_as_shipping` | pass | 배송 카테고리 회귀 가드 |
| `test_sent_negative_count_matches_filter_negative_all` | pass | Anchor에서 부정 0=0 (약한 통과) |

### Anchor 입력·기대 (TP-ANCHOR)

| 항목 | 값 |
|------|-----|
| 입력 | `"배송이 너무 늦어요. 화가 납니다."` |
| 기대 | `sent` 부정 1 · `filter(부정,배송)` ≥1 |
| As-Is | `sent` 중립 1 · `filter` 0건 |

### 회귀 입력

| 시나리오 | 입력 | Red 증상 |
|----------|------|----------|
| 중립 3건 | 보통이에요 / 괜찮아요 / 특별한 불만 없음 | sent 중립 2 ≠ filter 중립 1 |
| main only | `"품질"` | kw 1 ≠ filter 0 |
