## 구현 요약

- `parse_csv_content()` — `date,text` 헤더 시 `recorded_at` 파싱
- `trend_service` — 월(`YYYY-MM`)별 감정·카테고리 건수
- `HtmlRenderer` — **월별 추이** CSS 막대 차트
- 수동 `/analyze` — 날짜 없음 → 추이 섹션 미표시

## 데모 절차

1. `python app.py` → `http://localhost:8080`
2. `test_feedback_trend.csv` 업로드
3. **월별 추이**에서 2026-01~03 감정·키워드 막대 확인
4. `text`만 CSV 업로드 → 추이 없음, 기존 집계만 표시
